# 🧠 ThinkBoard

A full-stack MERN notes application where users can create, view, and delete notes easily.

## 🚀 Live Demo

👉 https://thinkboard-gcam.onrender.com

---

## 📌 Features

* ✍️ Create notes
* 📄 View all notes
* 🗑️ Delete notes
* ⚡ Fast and responsive UI
* 🌙 Modern UI with Tailwind CSS + DaisyUI

---

## 🛠️ Tech Stack

### Frontend

* React
* Vite
* Tailwind CSS
* DaisyUI

### Backend

* Node.js
* Express.js
* MongoDB
* Mongoose

## ⚙️ Installation (Local Setup)

### 1. Clone the repository

```bash
git clone https://github.com/coderashik247/thinkboard.git
cd thinkboard
```

### 2. Install dependencies

```bash
cd backend && npm install
cd ../frontend && npm install
```

### 3. Setup environment variables

Create a `.env` file inside backend:

```
MONGO_URI=your_mongodb_connection_string
PORT=5001
```

### 4. Run the project

```bash
npm run start
```
